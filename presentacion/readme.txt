Plantilla para presentación.

INSTRUCCIONES
=============

Compilar con lualatex o xelatex.

Utiliza la familia de tipografías Libertinus:

- Libertinus Serif
- Libertinus Math
- Libertinus Sans
- Libertinus Mono

pero puedes cambiarlo para usar las tipografías que quieras.

La bibliografía se lee por defecto del archivo bibliografía.bib, pero de nuevo, puedes cambiarlo por el que quieras.
